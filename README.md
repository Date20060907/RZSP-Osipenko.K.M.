## Запуск проекта
`gradle run`
## Документация
Собирается командой `gradle javadoc`
Найти можно в папке `app/build/docs/javadoc`
## Другое
В папке `docs` лежат блоксхема работы проекта и диаграмма на основе которой строилась БД
В папке `files_for_test` лежат файлы которые можно использовать при проверке программы (Файл со списком студентов и файл с дисциплиной)
